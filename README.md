# Proyecto final Módulo // Alke Wallet

Este proyecto correspondo al proyecto final del módulo 2 , su principal objetivo es el desarrollo de una Alke Wallet que pueda simular el funcionamiento de una billetera digital ( Login,Menu,Depósito,Transacciones,Envío de dinero).

## Tecnologías utilizadas // Alke Wallet

La Alke Wallet se cuenta con tecnologías como:

1. HTML5
2. CSS
3. JQuery
4. JavaScript
5. Bootstrap

## Funcionalidades principales

- Inicio de sesión de usuario (Login)
- Visualización del saldo disponible
- Depósito de dinero
- Envío de dinero a contactos
- Visualización del menú principal
- Persistencia del saldo utilizando Local Storage
- Mensajes de confirmación y alertas
